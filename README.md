# MERN User System – Frontend

A simple React frontend that connects to a Node.js + Express backend to register and login users.

---

## 🚀 Features

- User Registration Form
- User Login Form
- Axios API Integration
- Backend Connection Testing
- Error Handling

---

## 🛠️ Tech Stack

- React
- Axios
- JavaScript
- CSS

---

## 📂 Project Structure

```
user_system_frontend/
│
├── src/
│   ├── App.js
│   ├── components/
│   └── index.js
│
├── package.json
└── .gitignore
```

---

## 🔗 Backend Connection

The frontend connects to:

```
http://localhost:5000/api/users
```

Make sure backend is running before starting frontend.

---

## ⚙️ How To Run

1. Clone the repository
2. Run:
   ```
   npm install
   ```
3. Start frontend:
   ```
   npm start
   ```
4. Open:
   ```
   http://localhost:3000
   ```
